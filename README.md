# Spring Boot REST API - CRUD with MySQL (Course Management)

This project is a simple Spring Boot REST API that performs CRUD operations on **Course** using Spring Data JPA and MySQL database.

## Technologies
- Java
- Spring Boot
- Spring Data JPA
- MySQL
- Maven
- Postman (for API Testing)

## Database Setup
Create database:
```sql
CREATE DATABASE springboot_crud;
